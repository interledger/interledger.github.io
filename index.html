Simple Sending

Interledger makes it easy to send money, no matter what ledger you or the recipient are on.

  • Javascript
  • +
import { Client } from 'spp-client'

const client = new Client({
  username: 'alice@red.ilpdemo.org',
  password: 'secret'
})

function pay () {
  const payment = client.payment({
    destination: 'bob@blue.ilpdemo.org',
    destinationAmount: '0.01',
    memo: 'Still love you!'
  })

  return payment.send()
}

setInterval(pay, 1500)

Receiving Money

Interledger allows you to receive money from any ledger, without setting up accounts on lots of different services.

  • Javascript
  • +
  • see output
import { Client } from 'spp-client'

const client = new Client({
  username: 'bob@blue.ilpdemo.org',
  password: 'secret'
})

client.on('incoming', (payment) => {
  console.log('Received ' + payment.amount + ' bucks!')
  console.log(payment.sender + ' says: ' + payment.memo)
})
